Memory Card Game
A fun and interactive memory card game built with React and Vite. Test your memory by matching pairs of emoji cards in the fewest moves possible!

Features
Score Tracking: Keep track of your score and number of moves.
Win Condition: Celebrate when you match all pairs with a congratulatory message.
Reset Functionality: Start a new game anytime with the reset button.
Responsive Design: Optimized for desktop and mobile devices.
Shuffle Logic: Cards are randomly shuffled at the start of each game.

Technologies Used
React: For building the user interface and managing state.
Vite: For fast development and building.
CSS: For styling and animations.

How It Works
The game initializes with 16 cards (8 pairs of emojis) shuffled randomly.
Click a card to flip it. Click a second card to check for a match.
If they match, they stay flipped and your score increases.
If not, they flip back after a short delay.
The game locks input during animations to prevent rapid clicking.
Win when all pairs are matched.
